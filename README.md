# RSpec::Crispy

Custom matchers for RSpec to call [Crispy](https://github.com/igrep/crispy)'s API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rspec-crispy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rspec-crispy
